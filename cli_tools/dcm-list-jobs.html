dcm-list-jobs¶

List DCM jobs.

Description¶

Returns a list of DCM accounts. The list returned is subject to the scope of the API key.

Syntax¶

usage: dcm-list-jobs [-h] [--verbose] [--job-id JOB_ID]

optional arguments:
-h, --help            show this help message and exit
--verbose, -v         Produce verbose output
--job-id JOB_ID, -j JOB_ID Job ID

Options¶

Option Description
-v, –verbose Print out verbose information while listing jobs

Common Options¶

Deprecated Options¶

None

Output¶

The output from this command is a list of DCM jobs when no argument is passed, or if a job ID is passed, the return message from the job is provided.

Examples¶

Example 1¶

dcm-list-jobs
