dcm-list-users
--------------

List users.

Description
~~~~~~~~~~~

Returns a list of users.

Syntax
~~~~~~

.. program-output:: dcm-list-users -h

Options
~~~~~~~

+--------------------+-------------------------------------------------------------+
| Option             | Description                                                 |
+====================+=============================================================+
| -v, --verbose      | Print out verbose information while listing users.          |
+--------------------+-------------------------------------------------------------+

Common Options
~~~~~~~~~~~~~~

Deprecated Options
^^^^^^^^^^^^^^^^^^

None

Output
~~~~~~

The return value from this command is a list of DCM users.

Examples
~~~~~~~~

Example 1
^^^^^^^^^

.. code-block:: bash

   dcm-list-users
