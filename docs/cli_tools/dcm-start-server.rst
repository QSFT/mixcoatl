dcm-start-server
----------------

Start a server.

Description
~~~~~~~~~~~

Start a server that is in stopped/paused status.

Syntax
~~~~~~

.. program-output:: dcm-start-server -h


Output
~~~~~~

Job ID

Examples
~~~~~~~~

.. code-block:: bash

   dcm-start-server --server_id 47382

   54834
