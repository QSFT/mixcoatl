dcm-stop-server
---------------

Stops a server.

Description
~~~~~~~~~~~

Stops a server. The server will not be terminated but will be paused.

Syntax
~~~~~~

.. program-output:: dcm-stop-server -h

Output
~~~~~~

Job ID

Examples
~~~~~~~~

.. code-block:: bash

   dcm-stop-server 47382

   54833
