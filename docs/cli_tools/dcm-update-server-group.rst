dcm-update-server-group
-----------------------

Update the the owning group of a a server.


Syntax
~~~~~~

.. program-output:: dcm-update-server-group -h


Examples
~~~~~~~~

.. code-block:: bash

    $ dcm-update-server-group --server_id 95 --group_id 201
    Server with ID: 95 has changed to Group ID: 201
