dcm-update-volume-group
-----------------------

Update the the owning group of a a volume.


Syntax
~~~~~~

.. program-output:: dcm-update-volume-group -h


Examples
~~~~~~~~

.. code-block:: bash

    $ dcm-update-volume-group --volume_id 95 --group_id 201
    Volume with ID: 95 has changed to Group ID: 201
