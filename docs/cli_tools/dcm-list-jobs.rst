dcm-list-jobs
-------------

List DCM jobs.

Description
~~~~~~~~~~~

Returns a list of DCM accounts. The list returned is subject to the scope of the API key.

Syntax
~~~~~~

.. program-output:: dcm-list-jobs -h


Examples
~~~~~~~~

Example 1
^^^^^^^^^

.. code-block:: bash

   dcm-list-jobs
