dcm-detach-volume
-----------------

Detaches a volume from server.

Description
~~~~~~~~~~~

Detaches a volume from server.

Syntax
~~~~~~

.. program-output:: dcm-detach-volume -h

Options
~~~~~~~

+---------------------+-------------------------------------------------------+
| Option              | Description                                           |
+=====================+=======================================================+
| -v, --volumeid      | Volume ID of the volume to be detached.               |
|                     |                                                       |
|                     | Type: Integer                                         |
|                     |                                                       |
|                     | Default: None                                         |
|                     |                                                       |
|                     | Required: Yes                                         |
|                     |                                                       |
+---------------------+-------------------------------------------------------+

Output
~~~~~~

None


Examples
~~~~~~~~

.. code-block:: bash

   dcm-detach-volume -v 1105
