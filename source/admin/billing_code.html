mixcoatl.admin.billing_code¶

Implements access to the enStratus Billingcode API

class mixcoatl.admin.billing_code.BillingCode(billing_code_id=None, *args, **kwargs)[source]¶

Bases: mixcoatl.resource.Resource

A billing code is a budget item with optional hard and soft quotas against which cloud resources may be provisioned and tracked.

dict - Point where budget alerts will be triggered for this billing code
